# Features
Playground D++ is a software for stripped-down versions or lighter versions of D++ and config settings, which include D++ for x32 bit Systems

It Has the software depending on your specific needs:

- Software's(software folder, in the Software32-64-86 folder are all the softwares)
- **D++ For x32 Systems**, made for, yes for x32 Systems
- and more!
